Windows Nashville: The Forgotten Update

Explore the history of Windows "Nashville," the operating system intended to bridge the gap between Windows 95 and 98. Often rumored as "Windows 96," it pioneered the integration of the web into the desktop.

Section 1: What was Project Nashville?

üé∏

Started in late 1995, Nashville was planned as a minor update to Windows 95. While Windows 95 was a massive success, the internet was exploding in popularity, and Microsoft needed to catch up with Netscape.

The press and enthusiasts frequently referred to this project as "Windows 96", although Microsoft never officially confirmed this name. It was designed to run on the Windows 95 kernel but completely replace the user shell (the desktop interface).

Section 2: The "Web-Based" Desktop

üåê

Nashville's primary goal was to blur the line between your local hard drive and the internet.

  • Explorer is the Browser: In Nashville, the file explorer and the web browser were the same application. You could type a local path (C:\MyFiles) or a URL (http://www.microsoft.com) in the same address bar.
  • Single Click Navigation: To make files act like hyperlinks, Nashville introduced the option to open files with a single click (underlining icon titles when hovered), a feature that persists in Windows today.
  • Athena (PIM): A new "Personal Information Manager" intended to consolidate email, calendar, and contacts into one streamlined application.

Section 3: Cancellation & Strategy Shift

üõë
CHANGE OF PLANS: Nashville was cancelled as a standalone operating system release in late 1996.

Microsoft realized that releasing a whole new OS just for these features would confuse the market so soon after Windows 95. Instead, they decided to unbundle the features:

  1. Internet Explorer 4.0: The web integration code was packaged into IE 4.0.
  2. Windows Desktop Update: When users installed IE 4.0 on Windows 95 (and NT 4.0), it optionally installed the "Windows Desktop Update," which gave them the Nashville-style explorer.
  3. Windows 98 (Memphis): The remaining features and kernel updates were pushed to the next major project, codenamed "Memphis," which became Windows 98.

Section 4: What Survived? (Build 999)

The most famous leaked version is Build 999. It feels like a strange hybrid of Windows 95 and Windows 98.

Nashville Feature Evolution in Later Windows
Hover Select / Single Click This interface option survived and became a standard setting in Windows 98 (Folder Options).
"Athena" PIM This application evolved into Microsoft Internet Mail and News, which eventually became Outlook Express (for consumers) and full Microsoft Outlook (for Office).
Active Desktop The ability to put web widgets on the desktop wallpaper debuted here and was a flagship feature of Windows 98 (though it was resource-heavy and rarely used).
Fun Fact: While Windows Nashville was never sold, the codename appeared in the "Add/Remove Programs" list of early Internet Explorer 4.0 betas, identifying itself as "Windows Desktop Update (Nashville)."
