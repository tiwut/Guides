Linux Terminal Commands

A comprehensive reference guide for mastering the Linux Command Line Interface (CLI). From basic navigation to system administration.

Introduction

The Linux Terminal (Shell) is a text-based interface used to control the operating system. While modern Linux distributions offer Graphical User Interfaces (GUIs), the terminal provides unmatched power, speed, and the ability to automate complex tasks.

Most Linux systems use Bash (Bourne Again SHell) or Zsh as their default interpreter.

Navigation

Before you can modify files, you must know how to move around the file system.

Command Description
pwd Print Working Directory. Shows exactly where you are.
ls List directory contents.
cd [dir] Change Directory. Move into a folder.
Tip: Use ls -la to view hidden files (starting with a dot) and detailed permissions. 

# Go to the home directory
$ cd ~

# Go up one level
$ cd ..

# List all files with details
$ ls -la

File Operations

Essential commands for creating, moving, and deleting files and directories.

  • mkdir: Creates a new directory.
  • touch: Creates an empty file or updates timestamps.
  • cp: Copies files or directories.
  • mv: Moves or renames files.
  • rm: Removes files.

# Create a folder named 'projects'
$ mkdir projects

# Create a file inside that folder
$ touch projects/script.sh

# Copy a file (use -r for directories)
$ cp source.txt destination.txt

# Rename a file
$ mv old_name.txt new_name.txt
Danger Zone: The command rm -rf foldername forces the deletion of a directory and everything inside it. Deleted files are usually unrecoverable.

Permissions & Root

Linux is a multi-user system designed with security in mind. Access is controlled via Read (r), Write (w), and Execute (x) permissions.

Understanding Sudo

sudo (SuperUser DO) allows you to run commands with administrative privileges.


# Update system packages (requires root)
$ sudo apt update

# Change file permissions to be executable
$ chmod +x script.sh

# Change file owner
$ sudo chown user:group filename.txt

System & Networking

Tools to monitor system health and check connectivity.

Command Usage
top / htop View real-time CPU and Memory usage.
df -h Show disk space usage in human-readable format.
grep Search for specific text within files.
ip a Show IP addresses and network interfaces.
ping Check connection to a server (e.g., google.com).

Example of searching text inside a file:


# Find the word "error" in a log file
$ grep "error" /var/log/syslog
