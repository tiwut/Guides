macOS Productivity

Unlock the full potential of your Mac. This guide covers essential workflows, shortcuts, and system configurations to maximize your efficiency.

Overview

⚡

Productivity on macOS relies on reducing friction between your intent and the execution. By mastering navigation, file management, and system automation, you can save hours of work every week.

This guide focuses on native features that require no third-party software, ensuring these skills work on any Mac you touch.

Essential Keyboard Shortcuts

The keyboard is faster than the mouse. Memorize these core combinations to navigate the system rapidly.

  • Cmd + Space: Open Spotlight Search (Launch apps, calculate math, find files).
  • Cmd + Tab: Switch between open applications.
  • Cmd + ` (Tilde): Switch between windows of the same application.
  • Cmd + Shift + 4: Take a screenshot of a specific area.
  • Cmd + Option + Esc: Force Quit unresponsive applications.
Pro Tip: Hold the Option key while clicking distinct menus (like the Wi-Fi icon or Apple Menu) to reveal hidden details and diagnostics.

Window Management

Keeping your digital workspace organized is key to focus.

  • Mission Control: Swipe up with three fingers to see all open windows and create distinct Desktop Spaces.
  • Split View: Hover over the green maximize button to snap two apps side-by-side.
  • Hot Corners: Configure screen corners (System Settings > Desktop & Dock) to instantly lock the screen or reveal the desktop.

Spotlight & Finder Power

Stop digging through folders and start finding what you need instantly.

Did you know? You don't need to open a file to see it. Select a file and press Spacebar to use Quick Look.

Spotlight Tricks:

  • Type "100 USD to EUR" for instant currency conversion.
  • Type "weather" to see the forecast.
  • Drag a result from Spotlight directly into an email or document.

Terminal Tweaks

For advanced users, the Terminal allows you to change hidden settings to speed up animations or change default behaviors.

Example: Change the default screenshot file format from PNG to JPG (smaller file size) and disable the drop-shadow on screenshots:


# Change screenshot type to JPG
defaults write com.apple.screencapture type jpg

# Disable window shadow in screenshots
defaults write com.apple.screencapture disable-shadow -bool true

# Apply changes
killall SystemUIServer

Always verify commands before running them in your shell.
