AutoSheets

The ultimate automation tool for spreadsheet management, data processing, and reporting workflows.

Overview

üìä

AutoSheets is a high-performance framework designed to bridge the gap between raw data sources and presentation-ready spreadsheets. It allows developers and analysts to automate Excel and CSV tasks programmatically, removing the need for manual data entry and complex VBA macros.

Whether you are managing financial reports, inventory logs, or user databases, AutoSheets streamlines the pipeline from ingestion to final export.

Key Features

  • Universal Format Support: Seamlessly handle .xlsx, .csv, .ods, and Google Sheets formats.
  • Smart Formatting: Apply conditional formatting, cell styles, and formulas dynamically via code.
  • Data Validation: Automated error checking and data type enforcement before saving.
  • Performance: Process over 100,000 rows in seconds using the optimized C++ backend.
Note: AutoSheets requires Python 3.9 or higher to utilize the asynchronous processing modules.

Installation

Install the package via pip or your preferred package manager:


# Standard Installation
$ pip install autosheets

# Install with Cloud Connectors (AWS/Google Drive)
$ pip install autosheets[cloud]

Basic Usage

Here is a simple example of how to load a dataset, clean it, and export a formatted report:


import autosheets as asht

# Initialize the engine
engine = asht.Engine(mode='safe')

# Load and Process Data
doc = engine.load('raw_sales_data.csv')
sheet = doc.get_sheet(0)

# Automate: Remove duplicates and calculate totals
sheet.deduplicate(column='TransactionID')
sheet.add_column('Total', formula='=B2*C2')

# Apply styling to header row
sheet.rows[0].style(font='bold', bg_color='#CCCCCC')

# Export to Excel
doc.save('Monthly_Report.xlsx')
print("Automation Complete.")

Best Practices

To ensure data integrity and script reliability, adhere to the following guidelines:

Warning: Always perform operations on a copy of your production data until your scripts are fully tested.
  • Use Relative Paths: Avoid hardcoding absolute file paths to ensure your scripts work across different environments.
  • Error Handling: Implement try/except blocks to gracefully handle missing files or corrupt data entries.
  • Memory Management: For files larger than 500MB, use the stream_mode=True flag to process data in chunks.
